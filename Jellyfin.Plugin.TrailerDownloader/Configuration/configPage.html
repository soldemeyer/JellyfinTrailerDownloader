Trailer Downloader

Embedded yt-dlp is downloaded and managed automatically by the plugin and saves trailers directly next to your movies. The YoutubeDL-Material option queues downloads on a remote server — trailers only attach to movies if that server writes to the same media folders Jellyfin uses.
Both are native Jellyfin conventions. Movies sharing a folder with other movies always use the -trailer suffix.
Uses a "title year official trailer" search. Finds more trailers but can occasionally pick a wrong or fan-made video.
Placeholders: {title}, {year}

Trailer source priority

The plugin tries these sources in order until a trailer downloads successfully. AI is skipped unless an AI provider is configured below; YouTube search is skipped when the search fallback is disabled.

AI trailer search

Uses the provider's web search so the model finds real, current YouTube links to official theatrical trailers. If it finds multiple official trailers, all of them can be downloaded.
Leave empty for the provider default (Anthropic: claude-opus-4-8, OpenAI: gpt-5).

Automatic schedule

Saving this page sets the schedule of the "Download missing movie trailers" task (also visible under Dashboard → Scheduled Tasks, but this page overwrites it on save).

Embedded yt-dlp

Uses the community SponsorBlock database to cut flagged outros, end cards and self-promo segments during download. Only applies to videos where segments have been marked by the community.
Leave empty to let the plugin download and manage its own yt-dlp binary.
Only needed for age-restricted trailers that the automatic search retry can't recover. While signed in to YouTube, export a Netscape-format cookies.txt (e.g. with the "Get cookies.txt LOCALLY" browser extension), place it where the Jellyfin server can read it, and enter the full path here. Warning: cookies grant access to your YouTube account — keep the file protected.

YoutubeDL-Material server
